=========================================================
            DEVELOPER DEVELOPMENT & IPC GUIDE
                   Tái Sinh Ngọc Rồng Auto Tool
=========================================================

Tài liệu này giải thích luồng hoạt động nội bộ của code để Dev mới có thể đọc hiểu và phát triển thêm các tính năng In-game (Auto đánh quái, nhặt đồ...).

1. LUỒNG ĐĂNG NHẬP (AUTO LOGIN FLOW)
---------------------------------------------------------
(Xem file: `AutoLoginMod.cs`)

Game xử lý UI thông qua biến tĩnh (static). Mod sẽ can thiệp trực tiếp vào bộ nhớ để trigger:
B1. Đọc user/pass từ tham số dòng lệnh do Launcher truyền vào: `Environment.GetCommandLineArgs()`.
B2. Chờ cho màn hình `ServerListScreen` hiển thị. Tìm nút "Đổi tài khoản" (thông qua mảng `serverScr.cmd`) và gọi `performAction()`.
B3. Bảng `LoginScr` (Nhập nick) xuất hiện. Cần phải set cờ `LoginScr.isContinueToLogin = false` để xoá cache acc cũ.
B4. Gạch text vào ổ `loginScr.tfUser.setText()` và `tfPass`.
B5. Gọi `loginScr.savePass()` và cuối cùng là `loginScr.doLogin()`.

2. CƠ CHẾ IPC TỐI ƯU HÓA ĐA TAB (REAL-TIME CONFIG)
---------------------------------------------------------
Không giống như các Tool truyền thống tự móc hook Windows (WinAPI) để bóp handle game, Tool này sử dụng Kiến trúc IPC JSON file-polling trên Main Thread.

[LauncherApp] ----(Ghi file JSON)---> [AutoLoginMod_Settings.json] <---(Đọc file JSON mỗi 2 giây)---- [AutoLoginMod (In-game)]

Khi Dev click "Áp dụng" trên Launcher:
- Launcher parse Data thành chuỗi JSON và ghi đè vào file `Mods/AutoLoginMod_Settings.json`.

Khi Game chạy (`OnUpdate` trong `AutoLoginMod.cs`):
- Code kiểm tra `FileInfo.LastWriteTime` của file JSON.
- Nếu thời gian đổi -> Parse JSON để lấy Width, Height, FPS mong muốn.
- Gọi `Screen.SetResolution(w, h, false)`.
- *QUAN TRỌNG:* Unity gọi SetResolution chỉ thay đổi window size bên ngoài, còn tỷ lệ ma trận vẽ (Canvas Matrix) bên trong game Tái Sinh Ngọc Rồng là fix cứng. 
  Do đó, DEV bắt buộc phải hack trực tiếp vào các biến static `GameCanvas.w/.h/.hw/.hh` và gọi hàm private của game là `Il2Cpp.Main.main.setsizeChange()` để yêu cầu Game recalculate lại UI tỉ lệ thuận với Window Size.

3. MỞ RỘNG TÍNH NĂNG (GỢI Ý CHO DEV SAU)
---------------------------------------------------------
Nếu bạn muốn viết thêm Auto Làm Nhiệm Vụ, Auto Đánh Quái:
- Hãy sử dụng DnSpy/ILSpy để đọc class `Char`, `GameScr`, `Mob` trong file Dummy `Assembly-CSharp.dll`.
- Tạo thêm bảng cấu hình trong LauncherApp (Vd: Checkbox "Tự đánh", "Tự nhặt").
- Ghi trạng thái checkbox này vào `AutoLoginMod_Settings.json` để C# Mod đọc và thực thi hàm `GameScr.gI().autoPlay()`.
